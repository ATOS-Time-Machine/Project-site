--- title: Initial UI Design ---

Before working on the user interface we found it important that we have a design in mind, as it can take a lot of time to change in an coded prototype. We straight away went into developing a higher fidelity user interface prototype utilizing the Balsamiq wireframe software and skip making sketches as we could already partially base the design on what we saw in the user guide. We also made sure the designs that we were making were good examples of computer science and software engineering by making sure they were conforming to our requirements and the general HCI design principles.

  • Login
  • User Profile
  • Homepage
  • Overtime Request
  • Overtime Confirmation


The login page will be where the user can utilize their ATOS credentials to access “Time Machine”. They will also be able to recover their accounts if for some reason they don’t their password or it's the first time user the system since an administrator added them to the system.


The user profile page will be where users can view their profile details (+ profile details of those with a lower permission level) and change certain parts of it according to their role and assigned permissions that come with it, which can also be found in their profile.


The homepage will be where the users can find helpful information about how to navigate the website, and how to utilize its various features. There will tabs to make it intuitive for the users to find all of the information that they need.


The overtime request page will be where the users can access one of the main functionalities of this web application. On this page they will able to submit their in advance overtime request where they can clarify what there are going to be doing and why they need to do it in overtime.


The overtime confirmation page will be where approved overtime requests will be returned to the user after the date the specified the overtime would be occurring, and they can clarify what actually happened during this time, and whether it took the expected time.
